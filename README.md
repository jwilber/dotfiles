# dotfiles

Collection of my dotfiles for a better programming environment (on OSX).

## Installation
To install on your own machine, clone this repo and source the `install.sh`:

```
git clone https://github.com/jwilber/dotfiles.git
cd dotfiles
source install.sh
```
